# 플레이어 매뉴얼

## 게임 소개

반복되는 하루 속에서 동료와 가까워지고, 카드를 모아 덱을 강화하며, 위험한 맵을 돌파하는 로맨스 덱빌딩 로그라이크 게임이다. 선택은 대화, 전투, 엔딩에 영향을 준다.

## 안전 안내

- 이 게임은 15세 이상 이용을 기준으로 한다.
- 모든 로맨스 가능 캐릭터는 성인이다.
- 성인 캐릭터 간의 명시적 성적 콘텐츠는 일정 수위까지 허용된다.
- 로맨스는 설렘, 코미디, 상호 존중을 중심으로 표현된다.

## 기본 조작

- 마우스 클릭: 메뉴 선택, 대화 진행, 카드 사용, 노드 선택.
- Enter 또는 Space: 대화 진행.
- ESC: 일시정지 메뉴 열기 또는 닫기.
- 숫자 키 1-9: 전투 중 손패 카드 빠른 선택.
- R: 전투 중 선택 취소 또는 대상 지정 초기화.

## 타이틀 화면

- 새 게임: 처음부터 시작한다.
- 이어하기: 저장된 회차를 불러온다.
- 설정: 음량, 화면, 텍스트 속도를 조정한다.
- 갤러리: 해금한 엔딩과 추억을 확인한다.

## ESC 메뉴

플레이 중 ESC를 누르면 메뉴가 열린다.

- 계속하기: 게임으로 돌아간다.
- 저장하기: 현재 진행을 저장한다.
- 불러오기: 저장 데이터를 선택한다.
- 설정: 음량과 표시 옵션을 바꾼다.
- 타이틀로: 현재 회차를 나가 타이틀로 돌아간다.

## 덱 화면

- 현재 덱, 버린 카드, 획득한 유물을 확인한다.
- 카드 제거, 강화, 교체는 휴식 또는 특정 이벤트에서 가능하다.
- 카드 설명의 비용, 효과, 키워드를 확인하고 다음 전투를 준비한다.

## 맵 화면

- 각 노드는 전투, 이벤트, 휴식, 상점, 보스로 나뉜다.
- 연결된 다음 노드 중 하나를 선택해 이동한다.
- 위험한 경로일수록 강한 보상이나 특별 대화가 등장할 수 있다.

## 전투

1. 매 턴 손패를 뽑고 에너지를 받는다.
2. 카드를 선택한 뒤 대상이 필요하면 적이나 아군을 클릭한다.
3. 공격, 방어, 집중, 지원 효과를 조합해 턴을 운영한다.
4. 턴 종료를 누르면 적이 행동한다.
5. 승리하면 카드, 회복, 재화, 이벤트 보상 중 일부를 선택한다.

## 엔딩

엔딩은 전투 성과, 관계 수치, 중요한 선택, 기억 조각에 따라 달라진다.

- 캐릭터 엔딩을 보려면 해당 캐릭터와 꾸준히 대화하고 신뢰 선택지를 고른다.
- 진 엔딩은 여러 회차에 걸쳐 단서를 모아야 한다.
- 패배해도 일부 정보가 남아 다음 도전에 도움이 된다.
