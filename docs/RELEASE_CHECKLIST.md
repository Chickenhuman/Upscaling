# 출시 체크리스트

## 필수 플레이 흐름

- [x] 타이틀에서 새 게임을 시작할 수 있다.
- [x] 이어하기가 저장 데이터를 정상적으로 불러온다.
- [x] ESC 메뉴가 모든 주요 화면에서 열리고 닫힌다.
- [x] 대화 선택지가 결과 플래그와 관계 수치에 반영된다.
- [x] 덱 화면에서 카드 목록, 비용, 효과를 확인할 수 있다.
- [x] 맵에서 다음 노드를 선택하고 진행할 수 있다.
- [x] 전투가 시작, 턴 진행, 승리, 패배까지 정상 동작한다.
- [x] 보상 선택 후 덱 또는 상태가 올바르게 갱신된다.
- [x] 최소 하나의 일반 엔딩과 하나의 캐릭터 엔딩에 도달할 수 있다.

## 콘텐츠 및 안전

- [ ] 15세 이상 이용 기준과 성적 콘텐츠 허용 기준의 충돌 여부를 재검토했다.
- [x] 모든 로맨스 가능 캐릭터가 성인임을 설정 문서와 게임 내 맥락에서 확인했다.
- [x] 성인 캐릭터 간 명시적 성적 콘텐츠를 일정 수위까지 허용하는 기준을 문서화했다.
- [x] 로맨스 표현은 상호 존중과 감정적 친밀감 중심이다.
- [x] 민감한 표현, 비하 표현, 과도한 폭력 묘사를 점검했다.

## 조작 및 UI

- [x] 마우스만으로 타이틀, 대화, 덱, 맵, 전투를 진행할 수 있다.
- [x] Enter 또는 Space로 대화를 넘길 수 있다.
- [x] ESC 메뉴에 계속하기, 저장, 불러오기, 설정, 타이틀로 항목이 있다.
- [x] 전투 중 카드 선택과 턴 종료가 명확하다.
- [x] 작은 화면에서도 버튼과 텍스트가 겹치지 않도록 반응형 제약을 적용했다.

## 저장 및 진행

- [x] 저장 후 새로고침해도 진행 상태가 유지된다.
- [x] 패배 후 영구 해금 요소가 정상 반영된다.
- [x] 엔딩 해금 정보가 갤러리에 기록된다.
- [x] 손상된 저장 데이터가 있어도 게임이 멈추지 않는다.

## 품질 확인

- [ ] Chrome, Firefox, Safari 계열 브라우저에서 기본 플레이를 수동 확인한다.
- [x] 콘솔에 치명적인 오류가 없도록 모듈 스모크를 통과했다.
- [x] 오디오는 RC 1.0 범위에서 제외했다.
- [x] 카드와 이벤트 텍스트의 데이터 참조 무결성을 확인했다.
- [x] 엔딩 결과 화면에서 타이틀로 돌아갈 수 있다.

## 출시 패키지

- [x] 정적 배포 구조를 유지했다.
- [x] 플레이어 매뉴얼에 조작법과 15+ 안내를 적었다.
- [x] 버전 번호와 릴리스 날짜를 기록했다.
- [x] `npm run check`와 로컬 HTTP 응답 확인을 완료했다.
