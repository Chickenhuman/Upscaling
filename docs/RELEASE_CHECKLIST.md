# 출시 체크리스트

## 필수 플레이 흐름

- [x] 타이틀에서 새 게임을 시작할 수 있다.
- [x] 이어하기가 저장 데이터를 정상적으로 불러온다.
- [x] ESC 메뉴가 주요 화면에서 열리고 닫힌다.
- [x] 미연시 대화 화면에서 대사 진행과 선택지가 동작한다.
- [x] 전략 베팅 화면에서 말, 카드, 베팅 칩을 선택할 수 있다.
- [x] 레이스 결과 화면에서 순위, 적중, 지급/환급 결과를 확인할 수 있다.
- [x] 장 목표 달성 후 보상 카드를 선택할 수 있다.
- [x] 실패 엔딩과 캐릭터/팀/솔로 엔딩에 도달할 수 있다.

## 콘텐츠 및 안전

- [x] 15세 이상 이용 기준을 문서화했다.
- [x] 모든 로맨스 가능 캐릭터가 성인임을 확인했다.
- [x] README의 성적 콘텐츠 허용 문구를 유지했다.
- [x] 로맨스 표현은 설렘, 코미디, 상호 존중 중심이다.
- [x] 베팅은 현실 화폐와 연결되지 않는 허구 칩으로만 표현한다.

## 조작 및 UI

- [x] 마우스만으로 타이틀, 대화, 베팅, 레이스, 보상을 진행할 수 있다.
- [x] Enter 또는 Space로 대화를 넘길 수 있다.
- [x] 숫자 키 1-5로 전략 카드를 빠르게 선택할 수 있다.
- [x] ESC 메뉴에 계속하기, 저장, 불러오기, 덱, 설정, 타이틀 항목이 있다.
- [x] 작은 화면에서도 핵심 버튼과 텍스트가 겹치지 않도록 반응형 제약을 적용했다.

## 품질 확인

- [x] `npm run check`가 문법 검사, 스모크 테스트, 밸런스 시뮬레이션을 통과한다.
- [x] 5,000회 밸런스 시뮬레이션에서 승률이 목표 범위 안에 있다.
- [x] 생성 에셋 경로와 문서 경로가 스모크 테스트에서 검증된다.
- [ ] Chrome, Firefox, Safari 계열 브라우저에서 수동 플레이를 추가 확인한다.

## 출시 패키지

- [x] 정적 배포 구조를 유지했다.
- [x] 플레이어 매뉴얼과 게임 디자인 문서를 최신 구조에 맞췄다.
- [x] 개발 일지에 재개발 내용과 밸런스 결과를 기록했다.
