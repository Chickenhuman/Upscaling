# 게임 디자인 문서

## 개요

`네온 더비 로맨스`는 비주얼 노벨식 로맨스, 전략 카드 덱빌딩, 로그라이크식 실패/재도전 구조를 결합한 바닐라 JavaScript 웹 게임이다. 플레이어는 패덕 카페의 동료와 대화하고, 전략 카드를 조합해 야간 경주의 결과를 흔들며, 허구 칩 목표를 달성해 다음 장과 멀티 엔딩을 연다.

## 핵심 목표

- 미연시 대화 화면은 배경, 캐릭터 일러스트, 이름표, 하단 대화창이 분리된 전형적 VN 구성을 따른다.
- 덱빌딩은 전투 대신 레이스 전 분석 카드로 작동한다. 카드 선택은 점수, 단서, 환급, 지급 배율에 영향을 준다.
- 로그라이크 압박은 장별 칩 목표와 제한된 레이스 횟수로 만든다.
- 실패해도 엔딩 갤러리와 저장 데이터가 유지되어 다음 회차 동기를 남긴다.

## 안전 및 등급 기준

- 대상 연령은 15세 이상이다.
- 모든 로맨스 가능 캐릭터는 성인이다.
- README의 “명시적 성적 콘텐츠는 어느정도 허용한다” 기준을 따른다.
- 현재 구현 텍스트는 설렘, 코미디, 상호 존중을 중심으로 하며 현실 도박을 권장하지 않는다.
- 베팅은 현실 화폐가 아닌 게임 내 허구 칩으로만 표현한다.

## 주요 시스템

### 비주얼 노벨

- 7개 장의 대화와 선택지가 회차 진행을 이끈다.
- 선택지는 호감도, 칩, 말 단서에 즉시 반영된다.
- 최종 장의 선택과 누적 호감도에 따라 캐릭터, 팀, 솔로, 실패 엔딩이 갈린다.

### 전략 카드 덱

- 기본 8장으로 시작하고 장 클리어 보상으로 카드를 추가한다.
- 한 레이스마다 손패 5장 중 최대 3 포커스를 사용한다.
- 카드 계열은 단서, 안전, 출발, 스퍼트, 침착, 지구, 배당으로 나뉜다.

### 네온 레이스

- 5마리의 말과 5개 주로 조건이 순환한다.
- 말마다 출발, 지구, 스퍼트, 침착, 조건 적성이 다르다.
- 레이스 결과는 말 능력, 주로 조건, 단서, 전략 카드, 변동성으로 계산된다.

## 진행 흐름

1. 타이틀에서 새 게임 또는 이어하기를 선택한다.
2. 동행 캐릭터를 고르고 튜토리얼을 확인한다.
3. VN 대화를 읽고 선택지를 고른다.
4. 베팅 화면에서 말, 베팅 칩, 전략 카드를 선택한다.
5. 레이스 결과에 따라 칩, 단서, 호감도가 변한다.
6. 장 목표를 달성하면 보상 카드를 얻고 다음 장으로 넘어간다.
7. 최종 장에서 누적 관계와 선택에 맞는 엔딩을 본다.

풀 클리어 기준 예상 플레이타임은 68분이며, 스토리 125페이지와 대사 125페이지를 데이터 검증 대상으로 유지한다.

## 출시 기준

- 시작부터 엔딩까지 중단 없이 플레이 가능해야 한다.
- 저장, 불러오기, ESC 메뉴, 설정, 대화 진행, 전략 카드 선택, 레이스 결과가 안정적으로 동작해야 한다.
- `npm run check`는 문법 검사, 에셋/콘텐츠 스모크 테스트, 5,000회 밸런스 시뮬레이션을 모두 통과해야 한다.
