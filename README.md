# 네온 더비 로맨스

밝고 유쾌한 15+ 비주얼 노벨/전략 카드 로그라이크 웹게임이다. 정적 파일만으로 실행되며, 타이틀 화면, ESC 메뉴, 미연시 대화 화면, 전략 카드 덱, 허구 칩 기반 레이스 베팅, 보상, 저장/불러오기, 갤러리, 멀티 엔딩을 포함한다.

## 실행

```bash
npm run start
```

브라우저에서 `http://127.0.0.1:4173/`을 연다.

## 검증

```bash
npm run check
```

이 명령은 문법 검사, 콘텐츠/에셋 스모크 테스트, 5,000회 밸런스 시뮬레이션을 실행한다.

## 콘텐츠 기준

- 대상 연령은 15세 이상이다.
- 모든 로맨스 가능 캐릭터는 18세 이상의 성인이다.
- 명시적 성적 콘텐츠는 어느정도 허용한다. 하지만 설렘, 코미디, 상호 존중을 중심으로 한다.
- 풀 클리어 목표 플레이타임은 68분이며, 스토리 125페이지와 대사 125페이지가 회차 진행에 따라 분산 노출된다.
- 레이스 베팅은 현실 화폐와 연결되지 않는 허구 칩 진행 시스템으로만 표현한다.
